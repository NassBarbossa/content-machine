# LinkedIn Content Creation

Génère des posts LinkedIn à haute engagement.

## Sub-skills

Ce skill est divisé en 2 spécialisations :

| Skill | Fichier | Rôle |
|-------|---------|------|
| **Hook Generator** | `skills/hook/SKILL.md` | Génère 5 variantes de hooks |
| **Post Generator** | `skills/post/SKILL.md` | Génère le body à partir d'un hook choisi |

### Flow recommandé

```
1. Sujet → Hook Skill → 5 hooks générés
2. User choisit le meilleur hook
3. Hook choisi → Post Skill → Post complet
```

---

## Anatomie d'un post

```
┌─────────────────────────────────────┐
│ HOOK (1-2 lignes)                   │ ← Visible avant "...voir plus" (~210 car.)
│                                     │
│ ...voir plus ─────────────────────  │ ← Le gate du clic
│                                     │
│ BODY (story/valeur)                 │
│ - Une phrase par ligne              │
│ - Paragraphes courts (1-2 phrases)  │
│ - Listes ou points numérotés        │
│                                     │
│ CTA (1-2 lignes)                    │ ← Demande d'engagement
│                                     │
│ #hashtags (2-3)                     │
└─────────────────────────────────────┘
```

## Limites de caractères

| Élément | Limite |
|---------|--------|
| Post | 3,000 caractères |
| Visible avant "voir plus" | ~210 caractères |
| Hashtags | 2-3 recommandés |
| Commentaire | 1,250 caractères |

**Les 210 premiers caractères sont tout.** Si le hook rate, personne ne clique.

## Formules de Hook

### Ce qui marche

| Formule | Exemple |
|---------|---------|
| Opinion contrarian | "Opinion impopulaire : les code reviews sont une perte de temps." |
| Story personnelle | "J'ai été viré un mardi. Meilleure chose qui me soit arrivée." |
| Stat surprenante | "92% des startups échouent. Mais pas pour la raison que tu crois." |
| Promesse de liste | "J'ai recruté 200+ devs. Voici 5 red flags que je cherche." |
| Statement bold | "Ton CV ne compte pas. Voici ce qui compte." |
| Avant/après | "Il y a 3 ans je n'avais aucun entretien. Hier j'ai refusé une offre FAANG." |
| Pattern interrupt | "Stop. Avant d'envoyer ce cold email, lis ça." |

### Ce qui échoue

- ❌ "Ravi d'annoncer que nous sommes heureux de partager..." (corporate)
- ❌ "Dans le paysage en constante évolution..." (cliché, ne dit rien)
- ❌ "Je voudrais prendre un moment pour..." (lent, pas de hook)
- ❌ "Je viens de publier un article !" (pas de value prop)
- ❌ Commencer par un hashtag ou emoji

## Règles de formatting

### Les sauts de ligne sont tes amis

❌ Paragraphe dense :
"J'ai appris quelque chose d'important sur le leadership la semaine dernière. Mon équipe galérait avec une deadline et au lieu de pousser plus fort, j'ai décidé de réduire le scope. Le résultat était incroyable. on a livré plus vite et la qualité était meilleure."

✅ Formaté pour LinkedIn :
"J'ai appris quelque chose sur le leadership la semaine dernière.

Mon équipe galérait avec une deadline.

Au lieu de pousser plus fort, j'ai réduit le scope.

Le résultat ?

On a livré plus vite.
Et la qualité était MEILLEURE.

Parfois moins c'est plus."

### Guidelines

| Règle | Pourquoi |
|-------|----------|
| Une phrase par ligne | Plus facile à scanner sur mobile |
| Ligne vide entre paragraphes | Respiration visuelle |
| Paragraphes courts (1-2 phrases) | Lisibilité mobile |
| Sauts de ligne pour l'effet dramatique | Crée du rythme et du suspense |
| Bold sur phrases clés (avec parcimonie) | Attire l'œil sur les points importants |
| Listes numérotées pour les tips | Scannable, partageable |
| Éviter les murs de texte | Personne ne les lit |

## Types de posts (par engagement)

| Type | Engagement | Idéal pour |
|------|------------|------------|
| Story perso + leçon | Très élevé | Connexion, authenticité |
| Take contrarian | Élevé | Conversations, visibilité |
| Carousel (document) | Élevé | Contenu éducatif, tips |
| Liste/tips numérotés | Élevé | Valeur actionable, saves |
| Poll | Moyen-élevé | Engagement facile, data |
| Photo + story | Moyen | Humanisation, events |
| Vidéo native | Moyen | Démos, personnalité |
| Post avec lien | Faible | Traffic (algo pénalise) |
| Reshare | Très faible | Éviter. écris original |

### Stratégie liens

LinkedIn pénalise les posts avec liens. Solutions :

- **Méthode commentaire** : Post sans lien, ajoute le lien en 1er commentaire, édite le post pour dire "Lien en commentaire"
- **Méthode sans lien** : Résume le contenu dans le post, mentionne "DM pour le lien"
- **Si tu dois linker** : Mets-le à la toute fin, après du contenu standalone solide

## Signaux algorithme

| Signal | Impact | Comment |
|--------|--------|---------|
| Dwell time | Très élevé | Posts plus longs que les gens lisent en entier |
| Commentaires | Très élevé | Pose des questions, crée la discussion |
| Saves | Élevé | Contenu actionable, référençable |
| Clics "voir plus" | Élevé | Hook fort qui fait expand |
| Partages | Moyen | Contenu relatable, quotable |
| Réactions | Moyen | Facile à avoir mais moins pondéré |
| Liens externes | Négatif | Réduit la reach. liens en commentaire |
| Éditer après post | Négatif | Pas d'edit dans la première heure |
| Fréquence | 3-5x/semaine | Daily ok, plus d'1/jour pénalise |

## Schedule optimal

| Jour | Meilleur moment (timezone audience) |
|------|-------------------------------------|
| Mardi-Jeudi | 7-8h, 12h, 17-18h |
| Lundi | 8h (rattrapage) |
| Vendredi | 7-8h (avant déconnexion) |
| Weekend | Skip ou contenu léger |

**Engage dans les commentaires 30-60 min après avoir posté.** C'est plus important que le post lui-même.

## Formules CTA

| Type | Exemple |
|------|---------|
| Question | "C'est quoi le pire conseil carrière que tu as reçu ?" |
| Check d'accord | "D'accord ou pas d'accord ?" |
| Demande de partage | "Repost si ça résonne ♻️" |
| Prompt save | "Save ça pour ton prochain [situation] 🔖" |
| Demande de reco | "Tu ajouterais quoi à cette liste ?" |
| Demande d'expérience | "Ça t'est déjà arrivé ?" |

## Erreurs courantes

| Erreur | Problème | Fix |
|--------|----------|-----|
| Hook faible | Personne clique "voir plus" | Utilise les formules ci-dessus |
| Mur de texte | Illisible sur mobile | Une phrase/ligne, lignes vides entre |
| Liens dans le post | Algo réduit la reach | Liens en 1er commentaire |
| Trop de hashtags | Fait spam | 2-3 hashtags pertinents max |
| Jargon corporate | "Leveraging synergies" = scroll instant | Écris comme tu parles |
| Que de l'auto-promo | L'audience arrête d'engager | 80% valeur, 20% promo |
| Pas de CTA | Pas de direction d'engagement | Toujours finir par une question |
| Reshare sans ajouter | Reach quasi-nulle | Écris des posts originaux |
| Poster et disparaître | Tue le momentum des commentaires | Engage 30-60 min après |
| Être générique | "Le travail paie" = invisible | Stories et data spécifiques |

## Règles supplémentaires

- Pas de tirets longs "—" (pattern IA détectable)
- Metrics et anecdotes spécifiques > conseils génériques
- Expertise et opinion tranchée > contenu neutre consensuel
- Ton professionnel mais humain, avec personnalité
- Max 5 mentions/tags par post
